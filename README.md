# 九華 - Kyuka Online

花札を使った和風カジノ・カードゲーム「九華（くか）」のオンラインブラウザゲームです。

## 🎴 ゲーム概要

九華は花札を使用する対戦型カードゲームです。手札を揃えて「御免（あがり）」を目指します。

## 📋 基本ルール

- **使用札**: 1月から12月までの花札（各4枚、計48枚）＋ 特殊札2枚（計50枚）
- **手札**: 10枚からスタート
- **勝利条件**: 誰か1人が「御免（あがり）」を宣言した時点で1局終了。誰かのチップが0になった時点でゲーム終了

### ゲーム進行

1. 山札から1枚引く（手札11枚）
2. 手札から1枚捨てる（手札10枚）
3. 他のプレイヤーの捨て札で「頂戴（セット作成）」や「御免（あがり）」が可能

## 🌸 役（組み合わせ）

### 基本の形

| 役名 | 内容 |
|------|------|
| **三種（さんしゅ）** | 同じ月の札を3枚揃える |
| **対（つい）** | 同じ月の札を2枚揃える（あがりの「アタマ」として必要） |

### 特殊役

| 役名 | 組み合わせ内容 | 難易度 |
|------|----------------|--------|
| **三種（基本）** | 同じ月の札3枚 | ★☆☆ |
| **猪鹿蝶** | 7月（萩）・10月（紅葉）・6月（牡丹）の種札 | ★★★ |
| **赤短・青短** | 指定の短冊札3枚ずつ | ★★☆ |
| **月見酒・花見酒** | 8月（芒）または3月（桜）の光札 + 9月（菊）の杯 | ★★☆ |
| **九華（きゅうか）** | 1月から9月までの札を1枚ずつ揃える（究極役） | ★★★★★ |

### あがり条件（御免）

3セット（9枚）+ 1ペア（2枚）= 11枚で「御免」宣言

## 💰 チップと賭け（ベット）

| アクション | 説明 |
|------------|------|
| **積む（レイズ）** | 自分の番でチップを上乗せし、相手にプレッシャーをかける |
| **降りる（フォールド）** | 勝てないと判断し、その局の賭け金を放棄して降りる |
| **総取り** | あがったプレイヤーが場（ポット）に出ている全てのチップを獲得 |

## 🖐 割り込みアクション

| アクション | 説明 |
|------------|------|
| **御免（ごめん）** | 誰かの捨て札で役が完成した時の「あがり」宣言 |
| **頂戴（ちょうだい）** | 誰かの捨て札をもらって「三種（セット）」を完成させる |
