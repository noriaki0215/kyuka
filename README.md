# 九華 - Kyuka Online

花札を使った日本の伝統的なカードゲーム「九華（くか）」のオンラインブラウザゲームです。

## 🎴 ゲーム概要

九華は花札48枚を使用する対戦型カードゲームです。手札を揃えて「御免（あがり）」を目指します。

## 🎮 遊び方

1. **ゲーム開始**: 山札をタップしてゲームを開始します
2. **カードを引く**: 自分のターンで山札をタップして1枚引きます
3. **カードを捨てる**: 手札から1枚選んでタップして捨てます
4. **ターン交代**: CPUと交互にプレイします

### あがり条件

- **三種（さんしゅ）**: 同じ月の札を3枚揃える
- **御免（ごめん）**: 3セット（9枚）+ 1ペア（2枚）= 11枚であがり

## 🃏 花札について

12ヶ月の花札（各月4枚、計48枚）を使用します：

| 月 | 花 |
|---|---|
| 1月 | 松 |
| 2月 | 梅 |
| 3月 | 桜 |
| 4月 | 藤 |
| 5月 | 菖蒲（あやめ） |
| 6月 | 牡丹 |
| 7月 | 萩 |
| 8月 | 芒（すすき） |
| 9月 | 菊 |
| 10月 | 紅葉 |
| 11月 | 柳 |
| 12月 | 桐 |

## 🛠️ 技術仕様

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **レスポンシブ対応**: モバイルデバイスに最適化
- **サーバー不要**: ブラウザのみで動作

```

├── card.js       # カードデータ定義
├── logic.js      # 役判定ロジック
└── README.md     # このファイル
```

## 🚀 起動方法

1. リポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く

```bash
# ローカルサーバーで起動する場合（オプション）
npx serve .
# または
python -m http.server 8000
```
